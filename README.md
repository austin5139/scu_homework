### 1-2 當代計算機的通用架構
**當今計算機的通用架構，是基於馮紐曼模式(von Neumann Model)架構，此架構的主要 精神在於儲存程式(stored program)的概念。**
#### 馮紐曼模式主要有四大子系統
- 記憶體(Memory)
- 算術邏輯單元(Arithmetic Logic Unit；ALU )
- 控制單元(Control Unit)
- 輸入／輸出(Input／Output)
**算術邏輯單元和控制單元合起來稱為中央處理器(Central Processing Unit；CPU)**
